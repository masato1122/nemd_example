# nemd_example

Simple example for NEMD simulation

# Installation

python >= 3.0, ase, pymatgen

# How to Use

``` discriptions
cd ~/example
PYTHONPATH=$PYTHONPATH:../nemd
python mk_graphite_nemd.py -n 4 -m 3
```
